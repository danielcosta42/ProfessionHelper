# Profession Helper - Quick Start Guide

## ⚡ Quick Installation

### For Players
1. **Download** the latest version from:
   - [CurseForge](https://www.curseforge.com/wow/addons/profession-helper-tbc)
   - [GitHub Releases](https://github.com/danielcosta42/profession-helper-tbc/releases)

2. **Extract** the ZIP file

3. **Move** the `ProfessionHelper` folder to:
   ```
   World of Warcraft\_classic_\Interface\AddOns\
   ```

4. **Restart** World of Warcraft or type `/reload` in-game

5. **Open** the addon with `/ph` command or click the minimap icon

---

## 🎮 Basic Commands

| Command | Description |
|---------|-------------|
| `/ph` or `/professionhelper` | Open/close main window |
| `/ph show` | Show main window |
| `/ph hide` | Hide main window |
| `/ph farm` | Start farming session tracker |
| `/ph farm stop` | Stop farming tracker |
| `/ph guide <profession>` | Open gathering guide |
| `/ph help` | Show all commands |

---

## 🚀 First Steps

### 1. Open the Addon
- Type `/ph` in chat, OR
- Click the Profession Helper minimap icon

### 2. Select Your Profession
- Click any profession icon on the left sidebar
- Your learned professions are shown in color
- Unlearned professions are grayed out

### 3. View Your Path
- See step-by-step instructions for your current skill level
- Materials needed are listed for each step
- Click "Shopping" tab for complete material list

### 4. Use TSM Integration (Optional)
- Install TradeSkillMaster (TSM) addon
- Profession Helper will automatically show prices
- Click "Copy TSM Import" to create shopping groups

## 💡 Tips & Tricks

### Leveling Tips
- ✅ **Follow orange recipes first** - Always give skill points
- ⚠️ **Yellow recipes** - 75% chance for skill points
- 💚 **Green recipes** - Only 25% chance (use sparingly)
- ❌ **Grey recipes** - Never give skill points (skip!)

### Money-Saving Tips
- 💰 Look for **[Vendor]** tags - Buy from NPCs cheaply
- 🌿 Watch for **[Farm]** materials - Gather yourself
- 📊 Use TSM prices to find the cheapest route
- 🎯 Combine gathering + crafting professions

### Navigation Tips
- 🔍 Hover over profession icons for skill levels
- 📍 Use "Current" button to jump back to your skill level
- 🔄 Progress bars show your completion percentage
- 💬 Tips and tricks are shown for each step

## 🔧 Troubleshooting

### Addon doesn't show up?
1. Make sure folder is named exactly `ProfessionHelper`
2. Check it's in the correct AddOns folder
3. Enable "Out of date" addons at character select
4. Try `/reload` command

### No prices showing?
- Install TradeSkillMaster (TSM) addon
- Run a TSM desktop app scan
- Prices only show with TSM installed

### Professions not detected?
- Open your profession window (press 'K')
- Wait a moment for the game to update
- Reload UI with `/reload`

### UI looks wrong?
- Try adjusting UI scale in WoW settings
- Check display resolution
- Report the issue on GitHub
